# Pokédex con React y Vite

Este proyecto fue desarrollado durante mis prácticas de 1º de DAM en el **IES Gregorio Prieto**.  
La aplicación está construida con **React**, **Vite** y **JavaScript**, consumiendo la **PokéAPI** para mostrar información e imágenes de Pokémon.

## ✨ Características
- Consumo de API REST (PokéAPI).
- Renderizado dinámico de datos e imágenes.
- Navegación lógica entre Pokémon.
- Proyecto organizado con buenas prácticas en frontend.

## 🚀 Tecnologías
- React  
- Vite  
- JavaScript  
- PokéAPI
